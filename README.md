# Intruções sobre o Ambiente 
  * Banco de Dados - PostgreSQL 9.3
  * Linguagem de Progamação - Java 8.0_25
  * Container - Apache Tom Cat 8.0
  * IDE - NetBeans 8.0.2
  * Gerenciador de Depedência - Maven 4.0.0
